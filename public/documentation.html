Pet-Connect API Documentation

Server URL: http://149.200.187.89:55024/

Add User

Adds a new user account to the database

Endpoint: /users/addUser

HTTP Method: POST

Body Example:

"name": "Ahmad", 
"email": "ahmadaktaa@gmail.com", 
"password": "Ahmad123"

Successful Response Example:

"userID": "84796560982374039284"

Login User

Returns the userID when logging in with email/password

Endpoint: /users/login

HTTP Method: POST

Body Example:

"email": "ahmadaktaa@gmail.com", 
"password": "Ahmad123"

Successful Response Example:

"userID": "84796560982374039284"
