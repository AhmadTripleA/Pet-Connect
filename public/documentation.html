Register a New User

  • Endpoint: /users/register
  • HTTP Method: POST
  • Description: Registers a new user. All new users receive a default profile picture.
  • Parameters:
    • name (String): User's name.
    • email (String): User's email.
    • password (String): User's password.
    • phoneNumber (String): User's phone number.
  • Request Example: 
    {
    "name": "John Doe",
    "email": "johndoe@example.com",
    "password": "your_password",
    "phoneNumber": "123-456-7890"
}
  • Response:
    • Success (200 OK): 
      {
    "message": "User registration successful."
}
    • Error (4xx/5xx): 
      {
    "error": "Registration failed. Please check your input."
}

    Get User Profile Picture

    • Endpoint: users/getProfilePic
    • HTTP Method: GET
    • Description: Displays the user's profile picture.
    • Parameters:
      • email (String): User's email.
    • Request Example: 
      {
    "email": "johndoe@example.com"
}
    • Response:

      The response will be the user's profile picture in image format.

    Set User Profile Picture

    • Endpoint: users/setProfilePic
    • HTTP Method: POST
    • Description: Changes the user's profile picture. Use "multipart/form-data" encryption to send the request, including the email of the user and a new image.
    • Parameters:
      • email (String): User's email.
      • profilePic (Image File): New user profile picture.
    • Request Example: 
      {
    "email": "johndoe@example.com",
    "profilePic": "image_name.png"
}
    • Response:
      • Success (200 OK): 
        {
    "message": "Profile picture updated successfully."
}
      • Error (4xx/5xx): 
        {
    "error": "Profile picture update failed. Please check your input."
}

    Create a New Post

    Endpoint: /posts/newPost

    Method: POST

    Description: Create a new post in the database.

    Parameters:

    • email (String) - The email of the user who is the author of the post.
    • title (String) - The title of the post.
    • content (String) - The content or body of the post.
    • tags (Array of Strings) - An array of tags associated with the post for sorting and categorization.
    • images (Array of Strings) - An array of image locations associated with the post.
  • Request Example: 
    {
    "email": "user_email_here",
    "title": "New Post Title",
    "content": "This is the content of the new post.",
    "tags": ["tag1", "tag2"],
    "images": ["image1.jpg", "image2.jpg"]
}
  • Response:
    • Success (200 OK): 
      {
    "message": "New post created successfully."
}
    • Error (4xx/5xx): 
      {
    "error": "Failed to create the new post. Please check your input."
}
